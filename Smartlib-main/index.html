Fitur Utama Website Kami

Katalog Buku Lengkap

Cari, filter, dan jelajahi ribuan buku berdasarkan judul, penulis, kategori, atau status ketersediaan. Lihat detail buku dengan foto, deskripsi, dan rating.

Peminjaman & Pengembalian Online

Request pinjam buku dengan mudah. Admin akan konfirmasi, dan sistem otomatis update stok serta kirim pengingat tenggat via email.

Review & Rating Buku

Beri rating 1-5 bintang dan ulasan untuk buku yang telah Anda pinjam. Lihat rata-rata rating dan ulasan dari pengguna lain.

Notifikasi Otomatis

Dapatkan email pengingat 3 hari sebelum tenggat, notifikasi saat lewat tempo, dan info buku baru jika Anda subscribe.

Statistik & Backup

Lihat chart buku terpopuler dan peminjaman bulanan. Sistem otomatis backup data untuk keamanan.

Keamanan & Aksesibilitas

Role-based access (admin/member), sanitasi input, dan UI responsif untuk semua perangkat. Aman dari XSS dan unauthorized access.

Pelajari Dashboard Kami

Dashboard Anggota (Siswa)

Sebagai anggota, Anda dapat:

  • Melihat riwayat peminjaman lengkap dengan status (selesai, terlambat).
  • Mengakses katalog untuk mencari dan meminjam buku.
  • Memberi review dan rating pada buku yang telah dipinjam.
  • Menerima notifikasi email untuk pengingat tenggat.

Dashboard ini dirancang untuk kemudahan akses siswa SMK, dengan navigasi sederhana dan tampilan responsif.

Masuk sebagai Anggota

Dashboard Admin

Sebagai admin, Anda memiliki kontrol penuh:

  • Kelola buku: Tambah, edit, hapus dengan validasi input.
  • Kelola anggota: Lihat daftar dan aktifkan/nonaktifkan akun.
  • Lihat statistik: Chart buku terpopuler dan peminjaman bulanan.
  • Konfirmasi request peminjaman dan export riwayat ke CSV.

Dashboard admin dilengkapi sidebar untuk navigasi cepat, dengan akses role-based untuk keamanan.

Masuk sebagai Admin

Dashboard kami menggunakan teknologi Firebase untuk real-time updates dan Chart.js untuk visualisasi data yang menarik.
